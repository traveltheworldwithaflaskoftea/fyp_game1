1. Install requirements 
   ```
   $ pip install Flask-PyMongo
   $ pip install flask
   $ pip install jinja2
   ```
2. Run the Flask app in the terminal window.
   ```
   python app.py
   ```
   Navigate to http://localhost:5000/
